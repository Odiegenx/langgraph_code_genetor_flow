Create a small static website with a start page where the user can choose between three browser games: Snake, Tic Tac Toe, and Memory Match.

Snake Game:
- Real-looking snake and fruit using cartoon/CSS/canvas style (no external images).
- Snake eats apples, pears, and oranges that switch randomly.
- Arrow key controls that don't scroll the page during gameplay.

Tic Tac Toe Game:
- Play against AI with three difficulty levels: easy (child-friendly), medium (default adult), hard (optimal play).
- Tutorial menu with both static pattern explanations and interactive highlighting.

Memory Match Game:
- Two picture sets: fruits (default) and animals (switchable before game starts).
- Three grid sizes: 12, 24, and 48 cards.
- High scores stored in localStorage for each grid size.

Technical Requirements:
- Vanilla HTML, CSS, and JavaScript only.
- No external image/audio/video assets - use CSS, emoji, inline SVG, or canvas.
- Include a Node.js validation script that checks file structure and basic requirements.
