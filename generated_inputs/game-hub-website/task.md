# Static Game Hub Website

Create a small static website with a start page where users can choose between three browser games: Snake, Tic Tac Toe, and Memory Match. The site should use vanilla HTML, CSS, and JavaScript with no external dependencies.

## Requirements

1. Main landing page with navigation to three games
2. Snake game implementation
3. Tic Tac Toe game implementation
4. Memory Match game implementation
5. Consistent styling across all pages
6. Local validation script to check files and JS syntax
