# Generation Contract: Game Hub Website

## Dependencies
- Node.js (for validation only)
- No external libraries or frameworks
- No build tools required

## File Structure
```
project/
├── index.html          # Landing page
├── snake.html          # Snake game
├── tictactoe.html      # Tic Tac Toe game
├── memory.html         # Memory Match game
├── css/
│   └── styles.css      # Shared styles
├── js/
│   ├── snake.js        # Snake game logic
│   ├── tictactoe.js    # Tic Tac Toe game logic
│   └── memory.js       # Memory Match game logic
└── validate_site.js    # Validation script
```

## Technical Requirements
- Valid HTML5 markup
- CSS3 styling
- ES6 JavaScript
- No external dependencies
- Games must be fully playable
- All JavaScript must pass Node syntax validation
