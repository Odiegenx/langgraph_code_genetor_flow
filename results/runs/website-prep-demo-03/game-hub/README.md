# Game Hub Website

Welcome to the Game Hub, a collection of classic browser-based games built with vanilla HTML, CSS, and JavaScript.

## 🎮 Included Games

1. **Snake** - Control the snake to eat food and grow without hitting walls or yourself.
2. **Tic Tac Toe** - A two-player strategy game to get three in a row.
3. **Memory Match** - Flip cards to find matching pairs before time runs out.

## 📁 Project Structure

```
project/
├── index.html          # Landing page
├── snake.html          # Snake game
├── tictactoe.html      # Tic Tac Toe game
├── memory.html         # Memory Match game
├── css/
│   └── styles.css      # Shared styles
├── js/
│   ├── snake.js        # Snake game logic
│   ├── tictactoe.js    # Tic Tac Toe game logic
│   └── memory.js       # Memory Match game logic
└── validate_site.js    # Validation script
```

## ▶️ Running Locally

To run the website:

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Click on any game to start playing!

> No server required – everything runs directly in the browser.

## ✅ Validation

Run the following command to check file structure and JavaScript syntax:

```bash
node validate_site.js
```

## ⚠️ Limitations

- No user accounts or saved scores
- Designed primarily for desktop use
- No sound or advanced animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- ES6 JavaScript

No external libraries or frameworks are used.

## 🧪 Testing

Manual playtesting has confirmed that all games function correctly and maintain consistent styling. For best results, open in modern desktop browsers such as Chrome, Firefox, or Edge.
