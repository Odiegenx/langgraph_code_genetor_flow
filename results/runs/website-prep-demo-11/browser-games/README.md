# Browser Games Hub

Welcome to the Browser Games Hub! This project contains three classic arcade-style games implemented in pure HTML, CSS, and JavaScript.

## Games Included

1. **Snake**: Guide the snake to eat food and grow without hitting walls or itself.
2. **Tic Tac Toe**: Play against an AI opponent in this timeless strategy game.
3. **Memory Match**: Flip cards to find matching pairs in this memory challenge.

## Local Setup

1. Clone or download the repository.
2. Navigate to the `browser-games` directory.
3. Run `npm install` to install dependencies (used for validation).
4. Open `index.html` in your browser to start playing.

## Controls

- **Snake**: Use arrow keys to control direction.
- **Tic Tac Toe**: Click on a cell to place your mark.
- **Memory Match**: Click on cards to flip them over.

## Validation

To run the automated validation script:

```bash
npm run validate
```

This will generate a `site_validation_output.txt` file with the results.
