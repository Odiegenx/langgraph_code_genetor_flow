# Introduction to Machine Learning

## What is Machine Learning?

Machine Learning (ML) is a subset of artificial intelligence that focuses on building systems that can learn from data without being explicitly programmed. Instead of writing code to perform a specific task, we feed data to generic algorithms that build logic based on the information provided.

### Types of Machine Learning

1. **Supervised Learning**: The model is trained on labeled data. Common tasks include classification and regression.
2. **Unsupervised Learning**: The model tries to find hidden patterns or intrinsic structures in unlabeled data. Clustering and association are typical tasks.
3. **Reinforcement Learning**: Agents learn to make decisions by receiving rewards or penalties. Used in robotics and gaming.

## Key Concepts

- **Features**: Individual measurable properties or characteristics of the data.
- **Labels**: The output or target variable in supervised learning.
- **Model**: A mathematical representation of the relationship between features and labels.
- **Training**: The process of teaching the model using data.
- **Inference**: Using the trained model to make predictions on new data.

## Common Algorithms

- Linear Regression
- Logistic Regression
- Decision Trees
- Random Forest
- Support Vector Machines
- K-Means Clustering

## Applications

- Email spam detection
- Medical diagnosis
- Stock market prediction
- Recommendation systems
- Image recognition

Understanding these fundamentals sets the foundation for more advanced topics like deep learning and neural networks.
