# Workflow Task

Generate a small static website where the start page lets the user choose between three browser games:

- Snake
- Tic Tac Toe
- Memory Match

The project should be written with plain HTML, CSS, and JavaScript. TypeScript is acceptable only if the generated project also includes a no-dependency validation path; otherwise prefer vanilla JavaScript for reproducibility.

The workflow must demonstrate:

- architecture artifact generation
- ticket planning
- implementation by at least two worker roles
- multi-file website project generation
- validation execution
- automatic fix loop if validation fails
- quality report generation
- README and runbook generation
- deployment validation checklist
- final reviewer verdict
