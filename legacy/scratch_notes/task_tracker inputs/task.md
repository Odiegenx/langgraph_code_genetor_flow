# Workflow Task

Generate a small Java Maven task tracker CLI project and the required workflow artifacts for a local multi-LLM coding workflow evaluation.

The workflow must demonstrate:

- architecture artifact generation
- ticket planning
- implementation by at least two worker roles
- multi-file Java project generation
- Maven test execution
- automatic fix loop if tests fail
- quality report generation
- README and runbook generation
- deployment validation checklist
- final reviewer verdict
